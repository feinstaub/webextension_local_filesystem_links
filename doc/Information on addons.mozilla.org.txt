Name: Local Filesystem Links
Add-on URL: local-filesystem-links
Summary: Scans for local file system URIs (file:///) on any page. Click on the added icon to open the local link with Windows Explorer. Alternatively select any text representing an existing filesystem path like "C:\tmp\cow 1.txt" and use the context menu.
