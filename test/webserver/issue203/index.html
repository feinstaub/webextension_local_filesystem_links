If enable with option "Enable JavaScript support", file links can be opened with window.location.assign or window.open

File links

  • JS event (window.location.assign)
  • JS event (window.location.assign) - reveal folder
  • JS event (window.location.assign) - div element
  • JS event (window.open)
  • JS event (window.open) - div element

Folder links

  • JS event (window.location.assign) - folder
  • JS event (window.location.assign) - div element - folder
  • JS event (window.open) - folder
  • JS event (window.open) - div element - folder
